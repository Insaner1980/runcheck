1. BatteryHub (hyvin lähellä sinun ideaa)

Repo:
https://github.com/greenhub-project/batteryhub

Mitä se tekee:

monitoroi battery consumption

näyttää battery temperature

näyttää device specs

näyttää network info

näyttää storage ja memory usage

sisältää history charts 📈

Toisin sanoen:
se yhdistää battery + device info + monitoring samaan appiin.

Hyödyllistä sinulle:

Android battery API käyttö

historiallisen datan tallennus

graafit

Gradle-projekti

2. BetterBatteryStats (advanced diagnostics)

Repo:
https://github.com/asksven/BetterBatteryStats

Tämä on legendaarinen Android-diagnostics app.

Se tekee mm.:

löytää battery drainers

analysoi wakelocks

näyttää battery usage statistics

näyttää system power events.

Tämä projekti on erittäin hyödyllinen jos haluat:

nähdä miten syvä analytiikka tehdään

oppia battery stats parsing

ymmärtää Android power subsystemia.
